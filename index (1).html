📖 ¿Qué es el cómputo en la nube?

El cómputo en la nube (cloud computing) es un modelo tecnológico que permite acceder a recursos de cómputo — servidores, almacenamiento, bases de datos, redes y software — a través de internet, sin necesidad de contar con infraestructura física propia.

Entre sus principales proveedores se encuentran Microsoft Azure, Amazon Web Services (AWS) y Google Cloud Platform, los cuales ofrecen servicios escalables bajo demanda, reduciendo costos y facilitando el despliegue de aplicaciones a nivel global.

🔄 Proceso de la práctica
  • 1
    Elección de la implementación El equipo decidió desarrollar una página web hospedada en la nube, ya que permite demostrar el despliegue de contenido web de forma accesible desde cualquier dispositivo con internet.
  • 2
    Selección de plataforma: Microsoft Azure Se eligió Microsoft Azure como plataforma principal utilizando el servicio Azure Static Web Apps por su facilidad de despliegue e interfaz en español.
  • 3
    Intento de creación en Azure Se accedió al portal de Azure y se intentó crear un grupo de recursos llamado "practica_group" para alojar la aplicación web estática.
  • 4
    Bloqueo por política institucional Azure mostró el error: "La directiva no permitió el recurso practica_group" con la política "BloqueaTodaCreacionDeRecursos", impuesta por la Universidad Anáhuac, lo que impidió continuar en esta plataforma.
  • 5
    Solución alternativa: GitHub Pages Ante la restricción, el equipo optó por GitHub Pages como plataforma de hospedaje en la nube, igualmente válida para los objetivos de la práctica.
  • 6
    Despliegue exitoso Se creó el repositorio en GitHub, se subió el archivo index.html y se activó GitHub Pages, logrando publicar la página web en la nube de manera exitosa.
⚠️
¿Por qué no se pudo usar Microsoft Azure?

La cuenta de Azure de la Universidad Anáhuac tiene configurada una política de seguridad institucional llamada "BloqueaTodaCreacionDeRecursos", la cual impide la creación de nuevos grupos de recursos. Esta restricción es establecida por la administración universitaria para controlar el uso de recursos en la nube y evitar cargos no autorizados en la suscripción institucional.

✅
Solución implementada: GitHub Pages

GitHub Pages es un servicio de hospedaje web gratuito que publica sitios directamente desde un repositorio de GitHub. Al igual que Azure, AWS o Google Cloud, entrega contenido desde servidores distribuidos globalmente, siendo una alternativa válida de cloud computing para esta práctica.

🔗 Esta misma página es el resultado del despliegue exitoso en la nube.

📊 Resultados y conclusiones

Se logró desplegar exitosamente una página web en la nube utilizando GitHub Pages. La página es accesible desde cualquier dispositivo con conexión a internet, cumpliendo con el objetivo principal de la práctica: implementar un servicio en la nube.

La experiencia también permitió al equipo identificar y resolver un obstáculo real en entornos cloud: las políticas de seguridad institucionales, situaciones muy comunes en ambientes corporativos y educativos, y que forman parte del aprendizaje práctico en el área de cómputo en la nube.
